# SisoGuide 📚
A Django-based web platform for Class 11-12 students to access Previous Year Questions (PYQs) and Notes.

## Status
🚧 Work in Progress

## Planned Features
- User registration and login system
- Upload/Download PYQs for Class 11-12 CBSE/State boards
- Subject-wise and chapter-wise notes
- Search and filter functionality
- Admin panel to manage content
- Responsive UI for mobile and desktop

## Tech Stack
- **Backend**: Django, Python
- **Database**: SQLite / PostgreSQL  
- **Frontend**: HTML, CSS, Bootstrap, JavaScript
- **Deployment**: Coming soon

## Vision
To help students easily access organized study material and previous year papers in one place.
